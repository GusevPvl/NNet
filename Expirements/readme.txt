minHiddenLayers: минимальное количество скрытых слоёв (хотя бы 1 должен существовать)
maxHiddenLayers: максимальное количество скрытых слоев
minNeuronOnLayer: минимальное количество нейронов на скрытом слое
maxNeuronOnLayer:максимальное количество нейронов на скрытом слое
stepHiddenLayers: шаг изменения количества скрытых слоев
stepNeuronOnLayer: шаг изменения количества нейроной на скрытых слоях
combineNeuronsOnLayers: 0 - не комбинировать разные составы нейронов на слоях, 1 - комбинировать разные составы нейронов на слоях
trainingAccuracy: точность обучения, в программе считается как 1/заданнную здесь точность
maxtrainingAccuracy:максимальная точность обучения, в программе считается как 1/заданнную здесь точность
steptrainingAccuracy:шаг изменения точности обучения
trainingTimeLimit: ограничение времени на обучение в сек. (0-без ограничений)
intialDataType: 0 - DataBase, 1 - ExcelFile
observationsPerExpirement: количество запусков каждого эксперимента
biasOnOf: 0 или 1, нейрон смещения фактически не присутствует на предыдущем слое. Его видит только следующий слой и получает из него значения.
startMode: 0 - train, 1 - test
learningRate:100 - скорость обучения, определяется как 1/заданную скорость
maxlearningRate:максимальная(фактически минимальная) скорость обучения, определяется как 1/заданную скорость
steplearningRate: шаг изменения скорости обучения
epochCountLimit:0 - предел количества эпох обучения, 0 - без ограничений
fixedExpirement: 0 - стандартный запуск, 1 - слои берутся из файла FixedExpirement
newExpirementBook:0 - добавление в текущую книгу экспериментов, 1 - в новую
writeAllResults:0 - полная запись, 1 - только на 1 лист, 2 - 1 лист+результаты тестирования
